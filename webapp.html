Installera som webapp

Chrome på Android

  1. Öppna menyn ⋮.
  2. Välj Lägg till på startskärmen.
  3. Bekräfta genom att trycka på Lägg till.

Safari på iOS

  1. Tryck på delningsikonen.
  2. Välj Lägg till på hemskärmen.
  3. Tryck på Lägg till.

Chrome på dator

  1. Klicka på installationsikonen i adressfältet.
  2. Välj Installera.

Firefox

Firefox har begränsat stöd för webappar. Du kan skapa en genväg via webbläsarens meny.

Edge

  1. Klicka på menyn ⋯.
  2. Välj Appar > Installera webbplats som app.

Övriga webbläsare

Sök efter ett alternativ som Lägg till på startskärmen i webbläsarens meny.
